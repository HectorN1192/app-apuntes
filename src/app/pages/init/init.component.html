Fundamentos de angular

Que es angular?

Angular es un framework de desarrollo de aplicaciones web que permite crear aplicaciones de alto rendimiento y escalabilidad. Es un framework de JavaScript que utiliza componentes para construir la interfaz de usuario de una aplicación y proporciona un modelo de programación orientado a objetos para crear y mantener la lógica de la aplicación.

Componentes y Templates

Ciclo de vida de los componentes:

  1. Inicialización del componente.
  2. ngOnChanges: Responde a cambios en las
  3. propiedades de entrada.
  4. ngDoCheck: Detecta cambios manuales.
  5. ngAfterContentInit: Ejecutado después de que el contenido proyectado sea inicializado.
  6. ngAfterContentChecked: Llamado después de verificar el contenido proyectado.
  7. ngAfterViewInit: Llamado después de inicializar las vistas del componente.
  8. ngAfterViewChecked: Llamado después de verificar las vistas del componente.
  9. ngOnDestroy: Limpieza de recursos.

Creación y estructuración:

Uso de decorador Component. Definición de templates en línea y externos.

Directivas

Estructurales:

*ngIf, *ngFor, *ngSwitch.

Atributos:

Creación de directivas personalizadas con decorador Directive.

Binding de datos Interpolación: variable. Property binding: [property]="value". Event binding: (event)="handler()". Two-way binding: [(ngModel)]="variable".

Pipes

Uso de pipes integrados como date, currency, async. Creación de pipes personalizados con Pipe.

Inyección de Dependencias (DI)

Uso de Injectable para crear servicios. Proveedores en niveles de módulo y componente.

Módulos

Creación de módulos con NgModule. Configuración de AppModule, feature modules, y shared modules.
