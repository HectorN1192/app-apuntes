Fundamentos de angular

Que es angular?

Angular es un framework de desarrollo de aplicaciones web que permite crear aplicaciones de alto rendimiento y escalabilidad. Es un framework de JavaScript que utiliza componentes para construir la interfaz de usuario de una aplicación y proporciona un modelo de programación orientado a objetos para crear y mantener la lógica de la aplicación.
